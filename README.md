![Language](https://img.shields.io/badge/Python-yellow?logo=python&style=for-the-badge)

Popaction is a logic-driven movie ticket reservation application built to
demonstrate end-to-end system design under constrained resources.

This project focuses on application logic, data flow, and user experience
rather than production deployment or scalability.

---

## Features
- User authentication (signup, login, password reset)
- City-based movie selection
- Theatre and showtime selection
- Seat booking workflow
- Reservation history
- Multi-screen desktop UI

---

## Tech Stack
- Python
- CustomTkinter
- MySQL
- PIL / OpenCV

---

## Requirements
- Python 3.10+
- MySQL Server (local instance)

---

## Setup
1. Install MySQL
2. Create `config.json` using `config.example.json`
3. Add your local MySQL credentials
4. Run:
